# Travel Distance Roulette

A fun travel-inspired web application that lets users spin a roulette wheel to receive a random destination and then starts a countdown timer based on the estimated travel time to that location.

##  Features

-  Select a starting location
-  Spin a roulette wheel to get a random destination
-  Explore destinations across India
-  Simulated travel-time countdown
-  Responsive design for desktop and mobile
-  Fast and lightweight with no backend required
-  Every spin creates a unique travel challenge

##  Tech Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
